About

The Avenger Movie Database is a collaborative project for the class Software Engineering at the University of Zurich Institute for Computer Science. The aim is to take a provided source file and Stakeholder text and develop an interactable movie database according to their specifications.

Notes on the Use of the Application

In the following we would like to note some features of the application and inform about their limitations.

  • Display per Capita This button in the filter menu will display a worldmap using a selected dataset of a number of countries for comparison purposes. This dataset is not influenced by other filters as it is small due to data availability limitations. The formula is total movie releases divided by population times 100,000 (to achieve readable numbers).
  • Pie Chart The pie chart displays any information reduced by the chosen filters. Bear in mind though that when, for example, a filter for a certain country is set, it will still display other countries in the pie chart as many movies were produced in more than just one country.
  • Column Chart The Column Chart will not display the entire movie dataset as the visualization would simply become unreadable.
  • Table It is possible to sort the table alphabetically, according to release year or any other column by simply clicking on the column's header.
  • Worldmap There is a a limitation in the worldmap and per capita view on the side of the charts API, possibly related to certain browsers, in that it might take a while to draw all countries onto the map.
  • Timeline The timeline does not allow to increase the area the min and max year cover after it has been reduced. To increase the range, please use the Delete Chosen Filters button and set it anew.
